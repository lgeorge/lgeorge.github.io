The following list is ordered by Basic Program subject. If there is more than one course scheduled, they're listed chronologically within each subject. An "O" in the the "O/C" column means that a student can attend one or more individual sessions; "C" means the course is closed to new students after it starts. "Cert" indicates that completion of the course can fulfill the Basic Program requirement for the subject.
